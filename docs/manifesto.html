Research Question

Can numbers abstract anything?

CDQN explores this question by proposing Quang Numbers (Qn): finite computational abstractions capable of representing mathematical entities, semantics, operators and executable structures within a unified computational framework.

Vision

CDQN aims to bridge mathematics, physics and computer science through finite computational abstractions that remain executable on real hardware. Every abstraction is intended to be explicit, finite, traceable and validated.

Documentation as a Semantic Graph

The documentation is part of the architecture. Each HTML document is a node in the project graph. Hyperlinks express explicit relationships between specifications so that humans, browsers, AI agents and future CDQN tooling can navigate the same semantic structure.

HTML as an Interoperability Layer

Semantic HTML5 and the DOM are adopted as the bridge between contemporary software systems and the future CDQN runtime. Documentation is intentionally structured to be machine-readable as well as human-readable.

Lineage

This manifesto motivates the project. Formal definitions are provided by Qn Foundational Axioms. Subsequent specifications derive from those axioms.
