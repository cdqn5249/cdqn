Welcome

Welcome to the official documentation of the Chained and Distributed Quang Numbers (CDQN) project.

CDQN is an ongoing research and engineering project exploring a new computer science stack centered around Quang Numbers (Qn), Quang Mathematics (Qm), Quang Semantics (Qs), QnIR, QnLang, and the CDQN execution ecosystem.

Documentation Philosophy

The canonical documentation format of the CDQN project is semantic HTML.

Each HTML document defines exactly one concept.

All concepts are connected through explicit hyperlinks, forming a human-readable and AI-readable knowledge graph.
