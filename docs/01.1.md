## 1. Abstract: The Epistemic Disconnect

Contemporary computation and information theory operate on a "Flat Ontology" where numbers are treated as weightless, passive magnitudes. This paper argues that the current "Scalar Fallacy"—the belief that a number can be valid without a causal history or physical cost—is the root cause of systemic redundancy and the failure of machine "understanding."

We introduce the foundational atom of the **Chained and Distributed Quang Network (CDQN)**: the **Quang Number ($\mathcal{Q}$)**. Named after the Vietnamese concept of "Light/Clear" (Quang), this number system rejects abstract scalars in favor of **Constructive Triadic Relations**. We posit that by defining numbers through their ancestry and constraints, we can bridge the gap between Natural Language and Hardware Execution, creating a deterministic environment for biographical machine intelligence.

## 2. The Indictment: Why Legacy Mathematics Fails

### 2.1 The Redundancy of the Stateless Loop
In the legacy Turing-Von Neumann paradigm, a computer performs a task (e.g., calculating a video frame) and then "forgets" the geometric effort. If the user replays the task, the machine repeats the identical bit-logic cycles. This constitutes a terminal thermodynamic waste.
*   **Derivation:** Because legacy numbers lack **Ancestry**, the machine cannot recognize that it has already performed the work. It is trapped in a "Stateless Tape" where every instruction is a new, expensive event.
*   **Support:** **Landauer, R. (1961).** *Irreversibility and Heat Generation.* (Establishing that logical bit-erasures and redundant cycles incur a mandatory thermodynamic tax).

### 2.2 The Stochastic Hallucination of Flat Ontologies
Current Artificial Intelligence (AI) and Large Language Models (LLMs) treat "Context" as a set of probabilistic weights (floating-point decimals).
*   **The Failure:** Without a vertical umbilical cord to objective physical reality, these models "guess" at meaning. If a prompt is semantically consistent but physically impossible, the "Flat" math of the ALU cannot distinguish the error.
*   **Support:** **Berglund, L., et al. (2023).** *The Reversal Curse.* (Proving that 1D linear sequences fail to capture the bidirectional causal symmetries required for grounded reasoning).

## 3. Axiom I: The Triadic Construct $\{A \mid M \mid X\}$

We hypothesize that a number is not a point on a line, but a **Relation of Three Sets**. This is the fundamental unit of **Quang Mathematics**.

### 3.1 Definition of the Tuple
A Quang Number $\mathcal{Q}$ is a unified constructive object defined as:
$$\mathcal{Q} \equiv \{ \text{Ancestry } (A) \mid \text{Affinity } (M) \mid \text{Constraint } (X) \}$$

1.  **Ancestry ($A$):** The finite set of all historical states and metabolic seeds ($Q0$) that birthed this Quang. This provides the number's **Existence Proof**.
2.  **Affinity ($M$):** The internal directional vector or "Morphic Goal." This defines the number's **Semantic Meaning** (e.g., "Half," "Negative," "Move").
3.  **Constraint ($X$):** The set of all coordinates or rules this Quang physically/logically refuses to bond with. This provides the system's **Structural Honesty**.

### 3.2 Constructivist Origins
$\mathcal{Q}$-math draws inspiration from **Conway’s Surreal Numbers ($No$)**, where a number is defined by what it is "greater than" and "less than." 
*   **The CDQN Pivot:** We replace "Greater/Less" with **"Origin/Exclusion."** A number is defined by where it came from and what it is not.
*   **Finitism:** Unlike pure math, a Quang only exists if its triadic relation can be computed by a finite ALU in $N$ cycles. If it cannot be computed, it is **Null ($\emptyset$)**.
