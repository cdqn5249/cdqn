CDQN Documentation Standard

Version 0.1 — Accepted

Purpose

This document defines the standard structure for every canonical CDQN documentation page.

Every concept in the CDQN ecosystem shall be documented by exactly one semantic HTML document.

Documentation Invariant 001

Every concept shall have one canonical semantic HTML document, one stable identifier, one permanent URL, and one structured metadata block.
