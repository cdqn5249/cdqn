### 4. VON NEUMANN ERASER: BIAS CORRECTION

#### ✅ **STRONG: Classic Unbiasing Protocol (§5.1)**

**Algorithm:**
```markdown
1.  **Sample Pairs:** Take bit-pairs $(b_x, b_y)$ from StM-Loop-T
2.  **Logic:**
    *   If $(0,1)$, result = 0
    *   If $(1,0)$, result = 1
    *   If $(0,0)$ or $(1,1)$, discard and resample
3.  **Refined Stream $(S_t)$:** 50/50 probability bit-stream
```

**Mathematical Validity:**

**Theorem (Von Neumann, 1951):** Given biased bits with $P(0) = p$, $P(1) = 1-p$, the pairwise extraction produces unbiased output.

**Proof Sketch:**
$$P(\text{output}=0 \mid (0,1)) = P(\text{output}=1 \mid (1,0)) = p(1-p)$$

Since both outcomes have equal probability, output is unbiased regardless of input bias $p$.

**Efficiency Analysis:**

| **Input Distribution** | **Throughput Efficiency** |
|------------------------|--------------------------|
| $p = 0.5$ (unbiased) | 50% (half discarded) |
| $p = 0.9$ (heavily biased) | 18% (most discarded) |
| $p = 0.99$ (extreme bias) | ~2% (very wasteful) |

**CDQN Context:**
- **Cost acceptable:** Discarded pairs still consume Joules (metabolic work) ✅
- **Aligns with reputation model:** More work = more proof of existence ✅

**Verdict:** Algorithm choice is **theoretically sound** and **strategically appropriate** for CDQN's metabolic philosophy.

### **Recommendation 4: Clarify Clock Fraud Detection Threshold**

**Current (§4.2):**
> "If δ remains a constant for $N > 1000$ iterations, the machine identifies a **Clock Fraud Attack**"

**Ambiguity:** What constitutes "constant"?

**Possible Interpretations:**
1. **Exactly identical:** $\delta_i = \delta_{i+1} = ... = \delta_{i+1000}$ (statistically impossible on real hardware)
2. **Within small tolerance:** $|\delta_i - \bar{\delta}| < \epsilon$ for some threshold $\epsilon$
3. **Low variance:** $\text{Var}(\delta_{i...i+1000}) < \sigma^2_{min}$

**Suggested Clarification:**

```markdown
**Clock Fraud Detection Criterion:**

We define "constant" as **statistically improbable low entropy**:

$$\text{If } H(\{\delta_1, ..., \delta_{1000}\}) < 0.1 \text{ bits/sample} \Rightarrow \text{Clock Fraud Detected}$$

Where $H$ is estimated Shannon entropy over the 1000-sample window.

**Rationale:** Real hardware exhibits measurable jitter even under ideal conditions; near-zero entropy indicates virtualization or simulation.
```
