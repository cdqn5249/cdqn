# /cdqnLang

This directory contains the complete toolchain for **cdqnLang**, the sovereign, high-level programming language of the CDQN ecosystem.

The language is designed for safety, clarity, and verifiability, and transpiles to high-performance Rust.

### Components:
*   `/compiler` - The source for the cdqnLang -> Rust transpiler.
*   `/stdlib` - The language's standard library.
*   `/spec` - The formal language specification.
*   `/examples` - Example `.cdqn` source files.
