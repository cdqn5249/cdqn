# /src/core

The **Core (C-Layer)** contains the source code for the trusted services and infrastructure that make up the `cdqnRuntime`.

Unlike the Kernel, which defines universal truths, the Core layer implements the "business logic" of the system's operational nodes. This includes services for networking, state management, and inter-entity communication.
