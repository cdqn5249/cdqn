# /src/Shared

The **Shared (S-Layer)** is the designated space for shared, community-driven modules.

This layer is a critical part of the CDQN open core ecosystem. It contains reusable components and libraries that are not part of the trusted C-Layer core, but are designed to be used by multiple U-Layer applications.

The S-Layer promotes code reuse, accelerates development, and provides a formal pathway for community contributions to be integrated into the broader CDQN ecosystem. Modules in this layer may have their own distinct licensing to encourage community adoption.
