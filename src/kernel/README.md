# /src/kernel

This is the most sacred directory in the project.

The **Kernel (K-Layer)** contains the source code for the fundamental, immutable truths of the CDQN universe. These modules are designed to be minimal, pure, and have zero external dependencies. They form the verifiable foundation upon which the entire system is built.

### Modules:
*   `/hlc` - The Hybrid Logical Clock manager, the source of all causality and identity.
*   `/kdu` - The Kernel Data Unit factory and parser, for the system's atomic data structure.
*   `/entity` - The core logic for sovereign entities.
*   `/crypto` - The core cryptographic functions for signing and verification.
